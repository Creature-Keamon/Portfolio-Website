This Website

I built this website as to be used as a portfolio, where I can conveniently show all of my skills, any of my most interesting projects and any information you may (or may not) want to know about me. I built this website using HTML, CSS, JavaScript and ThreeJS, a JavaScript library.

I started working on this website in in December 2023, and I worked on it all the way through January and finished the core required parts of the website in February 2024. After I finished the core required parts I started working on adding all of my projects, about information and anything else that may be useful or informative. In June of 2024, I decided to make this website for visually appealing, I was going to give it a 3D flair. I had never touched ThreeJS, in my life, let alone JavaScript. But with the help of the ThreeJS Documentation and a video by the YouTube channel Fireship.

This website has been my first crack at doing absolutely anything with JavaScript, it's libraries and any build tools that I have used (Vite and NodeJS), and it certainly won't be my last

Watch this space! I will be documenting other projects that I work on here, updating ongoing ones and also updating the visuals of this website to better reflect my current skills with CSS.
