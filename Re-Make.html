Re-Make

Re-Volt is a racing game from 1999, and it is one of my favourite video games of all time. Of course, it shows it's age, and although some devoted fans have re-written it's ancient code so that it can take advantage of modern hardware and software frameworks like OpenGL, Vulkan and OpenCL, it still looks and sounds old. So I have started remaking the entire game from scratch, using the Unity game engine, C# programming language and remaking all of the models and assets in Blender and a yet to be decided 2D graphics program.

I started this project by creating a very basic vehicle controller that the player can use to drive a demo car. I then created a really basic track for the car to race on, and am currently learning how to create a checkpoint system, so that the game can track where the player is, when to give the player directions, and so that it can direct the AI cars correctly.
